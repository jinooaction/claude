# Contract — Judgment Points (헌법 III 계약)

각 판단 지점은 아래 다섯 가지를 코드 레지스트리(`judgment/registry.py`)로 선언한다.

## volatility_assessment (P1, US1)

| 계약 | 값 |
|------|-----|
| **트리거 조건** | 화이트리스트 종목의 `realized_vol_5m`(또는 동등 단기 변동성 통계) > 룰 임계값. 쿨다운 존중. 틱마다 호출 금지. |
| **입력 계약** | 요약 통계만: `{symbol, realized_vol_5m, threshold, recent_return_pct, ...}`. **원시 바 데이터 금지.** |
| **출력 스키마** | `{action: "hold"|"size_down"|"halt", confidence: float[0..1], reason: str}` |
| **지연 예산** | p95 < 2s. 초과 시 타임아웃 → 폴백. |
| **비용 예산** | $0.01/call. 롤링 초과 시 폴백 전환(FR-041). |
| **자본 영향** | 예 → 캐너리 5% 코호트, ≥10 거래일 (헌법 VI). |
| **소비(결정론적)** | `halt` & confidence ≥ `halt_min_confidence`(0.7) → 주문 건너뜀. `size_down` → `qty *= size_down_factor`(0.5). `hold` → 무효과. **노출 증가 불가.** |
| **폴백** | 자문 없음 = v1 동작(축소/건너뛰기 없음). 거래 진행. |

## daily_summary (P2, US2)

| 계약 | 값 |
|------|-----|
| **트리거 조건** | 장 마감 후 1회 (시각 게이트). |
| **입력 계약** | 그날 audit_log 집계 카운터(원시 행 아님). |
| **출력 스키마** | `{narrative: str(len≤500), alerts: list[str]}` |
| **지연 예산** | p95 < 10s. |
| **비용 예산** | $0.05/call. |
| **자본 영향** | 아니오 — 순수 자문. 주문 경로 무접촉. |
| **소비** | `auto-invest report --date <d>` 한 섹션으로 표시. |
| **폴백** | "요약 생성 불가", 결정론적 카운터만 표시. 리포트 정상 종료. |

## news_screen (P3, US3)

| 계약 | 값 |
|------|-----|
| **트리거 조건** | 장 시작 전, 화이트리스트 종목 매칭 헤드라인 **주입 시**. 공급원 없으면 비활성. |
| **입력 계약** | `{symbol, headline: str}`. |
| **출력 스키마** | `{stance: "bull"|"bear"|"neutral", confidence: float[0..1]}` |
| **지연 예산** | p95 < 5s. |
| **비용 예산** | $0.02/call. |
| **자본 영향** | 간접 — 신규 매수 보류만(노출 증가 불가). |
| **소비(결정론적)** | `stance == "bear"` & confidence ≥ `block_min_confidence`(0.8) → 그 종목 당일 신규 매수 보류. 그 외 무효과. |
| **폴백** | neutral 취급(거래 영향 없음). |

## 공통 불변량 (모든 판단 지점)

1. 출력은 **자문**, 주문 직접 제출 금지. 결정론적 게이트가 소비.
2. 자문은 **노출을 늘릴 수 없다** — 줄이거나 보류/건너뛰기만.
3. 같은 자문 → 같은 게이트 결정 (결정성, SC-002).
4. 매 호출: token_usage 1행 + LLM_CALL 1행, 같은 correlation_id. 본문·비밀 미기록 (헌법 V).
5. 실패/타임아웃/서킷/예산/스키마위반 → 결정론적 폴백. 거래 막지 않음 (SC-001).
6. 트리거+쿨다운으로만 호출, 틱마다 호출 금지 (헌법 III).
